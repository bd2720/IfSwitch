Tests the efficacy of if-else statements vs. switch/case statements when
each branch has the same likelihood of occurring.

COMPILATION:
	$ make
	$ ./if
	$ make ifSwitch

DESCRIPTION:
if.c/if: generates the if macro, "if.out"

if.out: if/else macro with N cases, generated by "if", included in "ifSwitch"

ifSwitch: runs and times each macro
